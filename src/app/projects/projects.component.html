Ticketing Site

Employee Management Interface

In the development of an Employee Management Interface, I employed an Angular, Tailwind CSS, and Angular Material for the frontend, along with the Node.js framework NestJS for the backend. This comprehensive CRUD application enables users to efficiently manage employee records through a user-friendly interface. The application allows users to create new employee records, read existing details, update information as needed, and delete outdated records.
